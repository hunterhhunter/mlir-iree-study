# 토이 튜토리얼

이 튜토리얼은 MLIR 위에 기본적인 토이(Toy) 언어를 구현하는 과정을 안내합니다. 튜토리얼의 목표는 MLIR의 핵심 개념을 소개하는 것이며, 특히 [방언](../../LangRef.md/#dialects)이 어떻게 언어 고유의 구성 요소와 변환을 쉽게 지원하면서도 LLVM 또는 다른 코드 생성 인프라로의 하향 경로를 간단하게 제공하는지를 다룹니다. 이 튜토리얼은 [LLVM 칼레이도스코프 튜토리얼](https://llvm.org/docs/tutorial/MyFirstLanguageFrontend/index.html)을 본떠 제작되었습니다.

추가로 참고하기 좋은 자료로는 2020 LLVM 개발자 회의의 온라인 [발표 영상](https://www.youtube.com/watch?v=Y4SvqTtOIDk)과 [발표 자료](https://llvm.org/devmtg/2020-09/slides/MLIR_Tutorial.pdf)가 있습니다.

이 튜토리얼은 MLIR을 내려받아 빌드한 상태를 전제로 합니다. 아직 수행하지 않았다면 [MLIR 시작하기](../../../getting_started/)를 참고하세요.

이 튜토리얼은 다음과 같은 장으로 구성되어 있습니다.

-   [1장](Ch-1_kr.md): 토이 언어 소개와 AST 정의.
-   [2장](Ch-2_kr.md): AST를 순회해 MLIR 방언을 생성하며 기본 개념을 소개. 여기서는 MLIR에서 사용자 정의 연산에 의미를 부여하기 시작합니다.
-   [3장](Ch-3_kr.md): 패턴 재작성 시스템을 활용한 고수준 언어별 최적화.
-   [4장](Ch-4_kr.md): 인터페이스를 사용한 일반적인 방언 독립적 변환 작성. 형상 추론과 인라이닝 같은 범용 변환에 방언 고유 정보를 연결하는 방법을 설명합니다.
-   [5장](Ch-5_kr.md): 더 낮은 수준의 방언으로 부분 하향. 고수준 언어 의미 중 일부를 최적화를 위한 범용 어파인 지향 방언으로 변환합니다.
-   [6장](Ch-6_kr.md): LLVM으로의 하향과 코드 생성. 코드 생성을 위해 LLVM IR을 대상화하고, 하향 프레임워크를 더 자세히 다룹니다.
-   [7장](Ch-7_kr.md): 토이 확장: 합성 타입 지원 추가. MLIR에 사용자 정의 타입을 추가하고 기존 파이프라인에 통합하는 방법을 보여줍니다.

[1장](Ch-1_kr.md)에서 토이 언어와 AST를 소개합니다.
